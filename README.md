# 👓 Google Glass — Projeto glass_html5

Este repositório contém o projeto prático desenvolvido durante o curso de **HTML5 e CSS3** da plataforma **Curso em Vídeo**, ministrado pelo professor **Gustavo Guanabara**.

O objetivo do projeto foi aplicar conceitos fundamentais de desenvolvimento web, criando um site completo e funcional para o dispositivo Google Glass.

---

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando as bases do desenvolvimento Front-End:

*   **HTML5:** Estruturação semântica do conteúdo.
*   **CSS3:** Estilização avançada, uso de Google Fonts, transições e efeitos de hover.
*   **JavaScript:** Implementação de funções de troca de ícones e cálculos dinâmicos no formulário.

---

## 📋 Funcionalidades

Durante o curso, foram implementadas as seguintes características:

*   **Menu Dinâmico:** Alteração de ícones ao passar o mouse via scripts JS.
*   **Layout Responsivo (Base):** Organização de elementos em grid e posicionamento CSS.
*   **Galeria de Fotos:** Exibição de imagens com legendas dinâmicas.
*   **Página de Especificações:** Uso de iframes para exibição de conteúdos interativos.
*   **Formulário de Pedido:** Validação de campos e cálculo de valor total em tempo real.
