#👓 Projeto Google Glass — HTML5 & CSS3
<br>
Este repositório contém o código-fonte do site Google Glass, um projeto prático desenvolvido durante o curso de HTML5 e CSS3 da plataforma Curso em Vídeo, sob a orientação do professor Gustavo Guanabara.

O projeto simula um site oficial para o dispositivo, focando em semântica, design clássico e interatividade básica.

#🛠️ Tecnologias e Conceitos Aplicados
O projeto foi o "campo de batalha" para aprender as bases sólidas do desenvolvimento web:

HTML5: Uso de tags semânticas (<header>, <nav>, <section>, <article>, <aside>, <footer>).

CSS3: Estilização com seletores avançados, efeitos de hover, transições, posicionamento (fixed, relative, absolute) e tipografia personalizada.

JavaScript: Manipulação básica de imagens para o menu dinâmico e cálculos automáticos no formulário.

Multimídia: Incorporação de áudio, vídeo e uso de iframes para conteúdos externos.

#🚀 Funcionalidades Demonstradas
Menu Inteligente: O ícone do menu principal muda dinamicamente conforme o usuário passa o mouse sobre os links.

Tabela de Dados: Organização técnica das especificações do Google Glass usando tabelas formatadas.

Galeria de Fotos: Exibição de imagens com legendas dinâmicas.

Formulário de Pedido: Inclui campos de data, cor, seleção de estado e um controle de quantidade que calcula o preço total em tempo real via JS.

Visualização 360°: Integração de conteúdo interativo para demonstrar o produto.
